# Web

あくあたん工房の公式Webサイト

## 実装するもの

* index.html
    * トップページ
    * コンタクトのリンクを貼る
* members.html
    * メンバー一覧
* 各メンバーのページ(`/members/hoge.html`)
    * 自分の属性（developerとかdesignerとか）
    * テンプレを一個作って各自で書く
* achievements.html
    * 実績一覧

## 個人ページの作り方

```bash
$ make mypage NAME="Your Name"
```
